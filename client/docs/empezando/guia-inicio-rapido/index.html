Guía de Inicio Rápido

Aprende cómo instalar y comenzar a usar el Orquestador de Lambda paso a paso.

Bienvenido a la Guía de Inicio Rápido del Orquestador de Lambda. Esta herramienta te permitirá generar proyectos estandarizados en segundos desde la consola. A continuación te mostramos los pasos para empezar a trabajar correctamente.

🧰 Requisitos previos

Antes de comenzar, asegúrate de tener instalado:

  • Python 3.10 o superior
  • Git
  • Un editor de código como Visual Studio Code
  • (Opcional) Un entorno virtual

1. 🚀 Instalar la librería

Puedes instalar la librería globalmente o dentro de un entorno virtual.

pip install git+https://github.com/LambdaAnalyticsSAS/LAMBDA_orquestador.git

Para verificar que se instaló correctamente:

orquestador version

Esto mostrará la versión instalada del orquestador.

2. 🧭 Crear un nuevo proyecto

Ejecuta el siguiente comando para lanzar la interfaz interactiva desde la consola:

orquestador crear

Esto abrirá un flujo paso a paso con las siguientes preguntas:

a. Seleccionar tipo de proyecto

1. Proyecto API basado en Django
2. Proyecto de Extracción, Transformación y Carga (ETL)
3. Proyecto de Automatización BOT

b. Nombre del proyecto

Reglas:

  • Sin espacios ni guiones normales (-), usar guiones bajos (_)
  • Sin tildes ni caracteres especiales
  • Mínimo 4 caracteres (sin contar guiones bajos)

c. Nombre de la empresa

Se utiliza como parte de la metadata del proyecto.

d. Autores del proyecto

Formato:

  • usuario1,usuario2
  • Usar nombres de usuario del correo corporativo (sin @lambdaanalytics.co)

e. Ruta de destino

  • Si se deja vacío, se usa la ruta actual.
  • Sin tildes ni caracteres especiales.

f. Descripción del proyecto

  • Mínimo 50 caracteres, máximo 200

g. Confirmación final

Confirma con y para generar el proyecto o n para cancelar.

3. 🛠️ Configuración según el tipo de proyecto

📦 Para proyectos Django:

cd nombre_proyecto
python -m venv venv
venv\Scripts\activate  # En Linux: source venv/bin/activate
pip install -r requirements.txt
# Editar el archivo .env con los datos de la base de datos
python manage.py migrate
python manage.py runserver

🧪 Para proyectos ETL o BOT

cd nombre_proyecto
python -m venv venv
venv\Scripts\activate  # En Linux: source venv/bin/activate
# Editar setup.py si estás desarrollando
pip install -e .  # Para desarrollo
# o
pip install .     # Para producción (código ya estable)

Inicia tu desarrollo dentro de la carpeta /pasos.

✨ Recomendación

Abre el proyecto con Visual Studio Code o tu editor favorito para comenzar a trabajar.

code .

¡Y listo! Ya puedes comenzar a desarrollar tu proyecto estandarizado con el Orquestador de Lambda.
